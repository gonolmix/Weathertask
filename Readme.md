# Погода по городу

Небольшое веб-приложение для получения актуальной информации о погоде.

Приложение использует **OpenWeatherMap API** и показывает текущую температуру, влажность, скорость ветра и описание погоды.

---

## Функционал

- Поиск погоды по названию города  
- Получение погоды по геолокации пользователя  
- Отображение температуры, влажности и скорости ветра  
- Поддержка русского языка   

---

## Что использовано

- **HTML5**
- **CSS3**
- **JavaScript**
- **OpenWeatherMap API**
